Universell Utforming og Search Engine Optimization

Universell Utforming

På denne nettsiden skal du bli mer kjent om Universal Utforming og Search Engine Optimization.

Universell Utforming kan lettest forklarer som «tilgjengelig og bra for alle». Det vil si at alt innholdene du har på nettsiden burde bli tilgjengelig for alle uavhengig om de er fargeblinde, bruker mobil eller laptop osv. For å implementere disse så må nettsiden være lett å forstå, bra kontrast og brukbar ved alle enheter.

En nettside bør ha en tittel som kan fort sier hva nettsiden egentlig handler om. Tekst font for å gjøre det lettere å lese innholdet. Bilde eller lyd for å gi brukeren lettere forståelse. Det er viktig å ha riktig headings og god struktur på nettsiden.

Search Engine Optimization

Search Engine Optimization kan hjelpe brukere til å finne nettsiden din lettere som Google eller Bing. Et bra eksempel er på SEO er når nettsiden kommer på første siden av Google. Når brukeren søker på en ting så er det mest og synlig at de kommer til å trykke på den øverste de se på første side. Måle til SEO er å lage ditt nettside slik at brukeren kan finne nettsiden før de andre.

Det er to type SEO, innenfor siden og utenfor siden. Innenfor siden vil være ditt innhold og hvordan du setter opp kode innenfor HTML. Ting som meta tags, alt tags for bilder kan påvirker nettsiden. Så det er viktig at det som stå i alt tags er relevante for nettsiden.

Utenfor siden vil være ting som sponsor, linka fra annet sider osv. Dette vil skape trafikk inni nettsiden og kanskje flere «vudering» og «troverdig» nettsider snakke om din nettside. Det går også an og bruke SEM (Search Engine Marketing) hvor man kan betale google for eksempel for å komme først og toppen på søkeren.

SEO står for Search Engine Optimization. For IT, så vil det si at man lager nettsiden så at folk som søker på search engine kan finne nettside vårt lettere
